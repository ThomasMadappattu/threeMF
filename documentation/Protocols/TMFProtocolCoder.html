TMFProtocolCoder Protocol Reference

Conforms to NSObject
Declared in TMFProtocolCoder.h

Overview

This protocol defines the contract for any encoder / decoder used within threeMF. Each coder must conform to this protocol and each peer communicating has to use the same encoding / decoding strategy in order to understand each other.
