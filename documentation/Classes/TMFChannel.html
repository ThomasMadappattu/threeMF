TMFChannel Class Reference

Inherits from NSObject
Declared in TMFChannel.h

Overview

Abstract class representing a network channel. A concrete channel communicates using a specific network technology (TCP sockets, HTTP rest API, …). Each peer has at least one channel but can have multiple for different types of commands.
