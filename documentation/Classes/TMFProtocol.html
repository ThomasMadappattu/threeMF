TMFProtocol Class Reference

Inherits from NSObject
Declared in TMFProtocol.h

Overview

Protocol responsible for parsing TCP / UDP data to TMFRequest and TMFResponse objects. Each peer talking has to use the same protocol and coder in order to talk to each other. Provide your own protocol by sub-classing and overwriting each public method.
