TMFDiscovery Class Reference

Inherits from NSObject
Declared in TMFDiscovery.h

Overview

This class handles the creation of a peer’s Bonjour services and the discovery of other peers on the network.

Tasks

  •   delegate

    The object that acts as the delegate of the receiving discovery. The delegate must adopt the TMFDiscoveryDelegate protocol. The delegate is not retained.

  •   configuration
