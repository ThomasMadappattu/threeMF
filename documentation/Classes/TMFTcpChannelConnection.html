TMFTcpChannelConnection Class Reference

Inherits from NSObject
Conforms to GCDAsyncSocketDelegate
Declared in TMFTcpChannelConnection.h

Overview

The class is used to handle a GCDasyncSocket connected. It represents a TCP connection from any TMFPeer in the 3MF peer 2 peer network. It is used to read TCP requests and sending responses.
